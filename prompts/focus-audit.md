# 焦點審計 Prompt（前期企劃用）

> **用途**：Solo Creator 在拍攝前，跟 Claude / ChatGPT 對話，把會分散主推焦點的內容**在企劃階段就過濾掉**（根本不用拍）。
>
> **比後期剪輯的價值**：省拍攝時間 + 省後期時間 + 不對的問題根本不要拍，比拍了再剪更乾淨。
>
> **真實案例**：Waffle House × byebyenoise! Tony 訪談 reel — 原腳本 6 Q，Q6 會失焦於主推 BBN，後期才剪掉。如果前期用這個 prompt，連拍都不用拍。

---

## Prompt（複製貼上用）

```
我有一個 [短片 / Reel / Vlog / 訪談 / 廣告] 的企劃，主推 [產品 / 主題 / 服務]。

這是我目前的問題清單 / 腳本大綱 / 分鏡列表：
[貼上你的腳本或大綱]

請扮演「焦點審計者」角色，逐項評估：

1. 每題 / 每段跟主推焦點的「關聯度」（1-5 分，1 = 完全無關，5 = 直接相關）
2. 移除後是否會傷敘事完整度？（會 / 不會 / 部分）
3. 如果保留，要做什麼修改才能對齊主推？

最後給我三個清單：
- ✅ **建議保留**：列出哪些，每項一句理由
- ❌ **建議刪除**：列出哪些，每項說明為何會失焦
- 🔄 **建議改題 / 改寫**：原版 → 新版，並說明對齊邏輯

審計重點：
- 商業焦點優先於內容完整度
- 寧可單一聚焦深，不要多元淺薄
- 識別「看似有趣但會分散注意力」的內容（這是最容易被誤留的）
```

## 使用範例（Tony 訪談 6Q 案例改寫對照）

### Input（Tony 訪談原腳本 6 Q）

```
我有一個音響評測訪談 Reel 的企劃，主推 byebyenoise! USB（電源穩壓器配件）。

問題清單：
Q1：音樂對你的意義是什麼？（文化 / 情懷 / 青春回憶 / 生活儀式感）
Q2：關於收藏黑膠這件事
Q3：對於黑膠跟串流音樂的看法
Q4：你的系統介紹（JBL Speaker / eversolo play 一體機）
Q5：byebyenoise! USB 對於系統提升的幫助
Q6：會推薦 eversolo play 給入門音響朋友嗎？升級前後系統最滿意的地方
```

### Expected Claude Output（理想版）

```
焦點審計結果：

✅ 建議保留：
- Q2 黑膠收藏（關聯度 4）：建立 Tony 是音響玩家身分，鋪陳 Q5 主推
- Q4 系統介紹（關聯度 4）：必要的背景脈絡，Q5 的對照基準
- Q5 byebyenoise 感受（關聯度 5）：**主推產品，是整支 reel 高潮**

❌ 建議刪除：
- Q1 音樂意義（關聯度 2）：太抽象，跟 BBN 沒有直接連結。建議改成「黑膠收藏」直接切入。
- Q6 eversolo 推薦（關聯度 2）：**會跟 Q5 BBN 競爭主推焦點**。eversolo 是另一個產品，觀眾會分心思考「兩個都要買嗎？」。砍掉讓 BBN 獨佔記憶點。

🔄 建議改題：
- Q1「音樂對你的意義」→「想要收藏黑膠唱片的原因」
  - 原版太抽象，新版直接拉到產品 angle 起點（黑膠 → 系統 → BBN）

審計重點達成：
- 6 Q 收斂為 4 Q（Q1 改 / Q2 / Q4 / Q5），主推佔比可拉到 50%+
- 砍 Q6 = 砍掉會分散 BBN 焦點的好內容（eversolo 推薦本身內容好，但會稀釋主推）
- 預估省 ~20 min 拍攝 + ~30 min 剪輯時間
```

## 何時用這個 Prompt

**最有 ROI 的時機（按優先序）**：

1. **業主提案前** — 客戶給你 reference 片 + 產品需求，你寫初稿腳本後跑審計再給客戶
2. **拍攝前最後一週** — 腳本確定但還沒拍，用 prompt 砍掉不必要的問題 / 鏡頭
3. **接案 / 個人專案企劃** — 任何包含「多個問題 / 多個產品露出 / 多個鏡頭」的企劃都適用

**不要在這個階段用**：
- 已經拍完了 — 用後期剪輯 SOP（會失焦的剪掉）
- 還沒寫腳本 — 先寫初稿再來審計，不要叫 Claude 從零生成腳本（那是另一個 prompt）

## 進階變體

### Variant 1：多產品競爭場景

```
我有 N 個產品要在這支 reel 裡露出 [列產品]。
請告訴我：
1. 應該選誰當主推（只能 1 個）
2. 其他產品如何當「環境元素」露出而不搶主推焦點
3. 哪些產品根本不該出現在這支 reel（留給其他支）
```

### Variant 2：訪談嘉賓篩選

```
我訪談嘉賓 [姓名 + 身分]，他想聊以下話題 [列表]。
我這支 reel 主推 [產品]，請告訴我：
1. 嘉賓的哪些話題能直接服務主推
2. 哪些話題即使有趣也要砍（會把訪談變成「嘉賓個人秀」而不是產品 reel）
```

### Variant 3：B-roll 取捨

```
我的 b-roll 素材清單：[列鏡頭]
這支 reel 主推 [產品]，預算 [秒數]。
請告訴我：
1. 必拍的 b-roll（支撐主推）
2. 可砍的 b-roll（看起來酷但不服務主推）
3. 缺什麼 b-roll 才能讓主推更完整
```

## 跟其他 Prompt 的關係

| Prompt | 階段 | 跟焦點審計的關係 |
|--------|------|----------------|
| **Hook 反推結構** | 研究（看別人 reel） | 焦點審計用研究結果決定**自己 reel 要不要保留某段** |
| **劇本拆解** | 企劃 | 焦點審計**接在劇本拆解之後**，刪掉劇本中失焦項 |
| **分鏡視覺化** | 視覺化 | 通過焦點審計的項目才進分鏡 |
| **焦點審計** | **企劃 → 視覺化之間的審計閘** | 守門員，決定什麼進到拍攝清單 |
