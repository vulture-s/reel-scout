# 劇本拆解 Prompt（把爆款骨架翻成自己鏡頭表）

> **用途**：拿到 Hook 反推結構抽出來的 4-beat 骨架後，把它翻成「我的題材鏡頭數 + 節奏 + 視角」的 shot list 草稿。
>
> **比直接寫鏡頭表的價值**：跳過「我該怎麼開頭」「應該幾鏡」這種卡關。Claude 根據骨架直接配鏡頭，你做選擇題而不是申論題。
>
> **位置**：研究（Hook 反推）→ **企劃（劇本拆解）** → 守門（焦點審計）→ 執行（分鏡視覺化）

---

## Prompt（複製貼上用）

```
我要把一個爆款骨架翻成自己 reel 的鏡頭表。

我的企劃資訊：
- 題材：[品牌形象 / 產品介紹 / 訪談 / 教學 / 紀錄 / 其他]
- 主推：[單一主推產品 / 主題 / 服務，盡量單一不要列三個]
- 平台：[IG Reel / TikTok / YouTube Short / YouTube long form]
- 目標長度：[15s / 30s / 60s / 90s+]
- 拍攝資源：[一人接案 / 兩人 / 完整劇組；自己出鏡 / 嘉賓出鏡 / 純 B-roll]

我從 Hook 反推抽出來的 4-beat 骨架：
1. Hook：[從研究階段抽出的 Hook 邏輯]
2. 反差介紹：[反差邏輯]
3. CTA：[行動呼籲設計]
4. 餘韻：[結尾留下什麼]

請按以下格式產出鏡頭表草稿：

## Act 結構（時間分配）
- Act 1（Hook）：[起 - 訖秒數]
- Act 2（反差介紹）：[起 - 訖秒數]
- Act 3（CTA）：[起 - 訖秒數]
- Act 4（餘韻）：[起 - 訖秒數]
（時間分配比例要符合該長度的節奏慣性 — 短影音 Hook 通常佔 15-25%，餘韻 5-10%）

## Shot List（逐鏡）

| # | 時間 | Act | 景別 | 焦段 | 視角 / 運鏡 | 內容 | 為什麼這樣拍 |
|---|------|-----|------|------|----------|------|----------|

針對每一鏡：
- 景別用 ELS / LS / FS / MS / MCU / CU / ECU 標記
- 焦段給具體數字（14mm / 24mm / 35mm / 50mm / 85mm / 135mm）
- 視角：eye level / low angle / high angle / POV / bird's eye / 其他
- 運鏡：static / pan / tilt / dolly / tracking / handheld / 其他
- 「為什麼這樣拍」一句話說明這一鏡的設計邏輯（不是「好看」是「服務 hook / 反差 / CTA / 餘韻 哪一塊」）

## 節奏建議
- 鏡頭數總量：[數字]
- 平均單鏡時長：[數字] 秒
- 最短鏡頭：第 [X] 鏡，[秒數]（hook 通常會有快剪）
- 最長鏡頭：第 [X] 鏡，[秒數]（餘韻通常會有 hold）
- 是否需要慢動作 / 快動作切換：[判斷 + 在哪些鏡]

## 拍攝資源檢查
根據我列的「拍攝資源」，告訴我這個鏡頭表：
- 一人接案能不能拍完？哪些鏡需要打光支援 / 第二人？
- 哪些鏡可以用 b-roll 替代主鏡（節省拍攝時間）？
- 哪些鏡是「不能省」的（拿掉就崩節奏）？

審計原則：
- 鏡頭數寧少勿多 — 短影音是節奏戰，不是訊息戰
- 每一鏡都要對應 4-beat 中的某一塊，不能「為了好看而存在」
- 一人接案企劃時，先標出「絕對核心 5 鏡」，其他都是 nice-to-have
```

## 使用範例（60s 幼兒園形象片）

### Input

```
我要把一個爆款骨架翻成自己 reel 的鏡頭表。

我的企劃資訊：
- 題材：品牌形象（幼兒園）
- 主推：某家幼兒園的「特色課程 + 溫暖質感」
- 平台：YouTube + IG Reel 雙投放
- 目標長度：60s
- 拍攝資源：一人接案，DP 一人；園方配合，小朋友自然鏡頭優先

骨架（從 Hook 反推抽出來，題材轉譯版）：
1. Hook：打破「幼兒園 = 制式介紹」預期 — 用小朋友自己的視角開場（POV / 低機位）
2. 反差介紹：不是課表羅列，是「課程在做什麼 + 小朋友的反應」並置
3. CTA：招生 / 參觀預約（不要硬推，用情感包覆）
4. 餘韻：園長或老師抱小朋友的暖鏡頭 + 園所 logo 慢慢淡入
```

### Expected Claude Output（理想版）

```
## Act 結構（60s 分配）
- Act 1（Hook）：0-8s（13%）— 小朋友 POV 視覺，打破預期
- Act 2（反差介紹）：8-35s（45%）— 課程主軸（4-5 鏡分配給 2-3 個課程）
- Act 3（CTA / 特色場景）：35-50s（25%）— 招生 hook + 環境特色
- Act 4（餘韻）：50-60s（17%）— 暖鏡頭收尾 + logo

## Shot List

| # | 時間 | Act | 景別 | 焦段 | 視角 / 運鏡 | 內容 | 為什麼這樣拍 |
|---|------|-----|------|------|----------|------|----------|
| 1 | 0-2s | Hook | ECU | 35mm | POV / handheld | 小朋友手伸向草地泡泡 | Hook：第一秒非預期視角，觀眾意識到「這不是大人觀點」 |
| 2 | 2-5s | Hook | LS | 24mm | low angle / tracking | 小朋友奔跑穿過走廊 | 延續低視角，建立「孩子主場」基調 |
| 3 | 5-8s | Hook | MS | 35mm | eye level / static | 小朋友看鏡頭微笑 | Hook 結尾轉介人物，準備進反差介紹 |
| 4 | 8-15s | 課程 | MS + ECU 切換 | 50mm | static + ECU pan | 魔法廚房：打蛋過程 + 小朋友驚訝臉部特寫 | 反差：不講「我們有烹飪課」，給「打蛋瞬間 + 反應」 |
| 5 | 15-22s | 課程 | FS | 35mm | high angle | 蛋液成形俯拍 + 小手伸入 | 反差延伸：成果感，俯拍 abstract beauty |
| 6 | 22-28s | 課程 | MCU | 85mm | eye level | 小朋友吃成品，笑容特寫 | 反差收尾：情感記憶錨點，主推「孩子在這裡開心」 |
| 7 | 28-35s | 課程 B-roll | 快剪 4-5 短鏡 | 35mm + 85mm | 多視角 | 小雞 / 武功 / 高爾夫 / 平板教學 quick cut | 用 b-roll 補課程多樣性，避免單一課程獨大 |
| 8 | 35-42s | 特色 | LS | 24mm | tracking | 園內走廊全景，老師牽小朋友 | CTA 鋪陳：環境感 + 師生互動 |
| 9 | 42-50s | 特色 | MCU | 85mm | low angle | 老師蹲下跟小朋友對話 | CTA 強化：信任感，園方人格特質 |
| 10 | 50-55s | 餘韻 | CU | 85mm | eye level / slow zoom | 園長 / 老師抱小朋友 | 暖鏡頭：情感收斂 |
| 11 | 55-58s | 餘韻 | MS | 35mm | static | 小朋友揮手 + 園所遠景 | 從近到遠的視覺收場 |
| 12 | 58-60s | Title | Title Card | — | static | 園所 logo + slogan + 招生 / 預約資訊 | 收尾 CTA |

## 節奏建議
- 鏡頭數總量：12 鏡（短影音慣性 60s 通常 10-14 鏡）
- 平均單鏡時長：5s（含 4-5 鏡 quick cut 段 ~1.5s/鏡）
- 最短鏡頭：第 7 鏡內的 quick cut，每個 ~1.5s（製造節奏起伏）
- 最長鏡頭：第 4 鏡，7s（魔法廚房需要時間敘事）
- 慢動作建議：第 6 鏡（小朋友吃成品 0.7x slow-mo）+ 第 10 鏡（抱小朋友 0.5x）

## 拍攝資源檢查
一人接案 + DP 一人可以拍完，但要注意：
- 第 1, 2 鏡（POV / low angle handheld）：DP 主拍，你做 director + 小朋友互動引導
- 第 4 鏡（打蛋過程）：需要兩機位（廣 + 特），如果只一人要分兩 take 拍
- 第 6 鏡（吃成品笑容）：是「不能省」鏡，要等到孩子真實反應出現才拍，準備 10 分鐘以上
- 第 7 鏡（4-5 b-roll quick cut）：可以分兩天拍，不需要連場
- 第 10, 11 鏡（餘韻）：要在最後拍，演員不能累垮

絕對核心 5 鏡：1（hook 視角）、4（打蛋反差）、6（笑容情感）、10（暖收尾）、12（CTA card）。其他都是 nice-to-have，現場 time 不夠先砍 7（b-roll 補課程）或 9（CTA 強化）。
```

## 何時用這個 Prompt

**最有 ROI 的時機**：
1. **跑完 Hook 反推、進企劃前** — 骨架抽完，要落地成可執行清單
2. **業主給 reference 後** — 業主丟一支 reel 說「我要這種感覺」，你需要快速產可報價的鏡頭表
3. **題材跨界** — 拍習慣的題材換到新領域（例如習慣拍美食換拍人物），先用這個 prompt 對齊節奏感

**不要在這個階段用**：
- 連 4-beat 骨架都還沒抽 — 先跑 Hook 反推
- 主推超過 1 個 — 先跑焦點審計（這份 prompt 假設主推已收斂）

## 進階變體

### Variant 1：訪談類 reel（嘉賓 + 主題）

```
我的企劃是訪談類 reel，不是純拍 b-roll。

嘉賓資訊：
- 嘉賓身份：[姓名 + 一句話定位]
- 拍攝場景：[嘉賓家 / 工作室 / 戶外 / 棚拍]
- 嘉賓 vs 主推產品的關係：[嘉賓是用戶 / 嘉賓是專家 / 嘉賓是品牌大使 / 其他]

請按訪談 reel 的特殊節奏處理：
- A-roll（嘉賓口述）：通常 50-70% 時間
- B-roll（環境 + 產品 + 嘉賓動作）：30-50%
- 主推產品出現次數：不要超過 3 次（避免變廣告）
- 嘉賓出鏡多少時間：建議 50% 以上（讓觀眾跟嘉賓建立連結再帶到產品）

其他比照主 prompt 處理。
```

### Variant 2：純產品 reel（沒有人）

```
我的企劃是純產品 reel，沒有人物出鏡。

產品資訊：
- 產品型態：[實體 / 數位 / 服務 / 抽象概念]
- 視覺特徵：[顏色 / 質感 / 大小 / 動態]
- 使用情境：[誰在什麼場合用]

純產品 reel 的鏡頭表特殊規則：
- Hook 通常是「產品的非預期使用 / 視覺奇觀」
- 反差介紹會著重「產品 vs 競品」或「使用前 vs 使用後」
- CTA 需要強化（沒人物代言所以要直接）
- 餘韻通常是產品在環境中的 mood shot

特別注意：
- 一定要列「能不能在家自拍 vs 需要工作室」的判斷
- 純產品鏡頭要列「光線需求」（柔光 / 硬光 / 環境光），這比人物鏡頭嚴格

其他比照主 prompt 處理。
```

### Variant 3：B-roll only 教學類

```
我的企劃是 b-roll only 的教學類影片（旁白後配，不出鏡）。

教學主題：[一句話描述教學內容]
旁白風格：[輕鬆 / 嚴肅 / 反差 / 故事感]

特殊節奏：
- 鏡頭服務旁白，不是旁白服務鏡頭
- 同一個 b-roll 可重複出現（建立視覺 motif）
- 必須有 2-3 個「視覺記憶點」（旁白記不住但畫面記住的）

其他比照主 prompt 處理。
```

## 跟其他 Prompt 的關係

| Prompt | 階段 | 跟劇本拆解的關係 |
|--------|------|---------------|
| **Hook 反推結構** | 研究 | 產出本 prompt 的「4-beat 骨架」輸入 |
| **劇本拆解**（本檔） | 企劃 | 把骨架變成鏡頭表 |
| **焦點審計** | 企劃 → 視覺化之間 | 審視鏡頭表是否對齊主推（砍會分散注意力的鏡） |
| **分鏡視覺化** | 視覺化 | 通過焦點審計的鏡頭，逐鏡寫生成 prompt |

順序：**Hook 反推 → 劇本拆解 → 焦點審計 → 分鏡視覺化**

---

## 一人接案的額外提醒

Solo Creator 用這個 prompt 時，特別注意 Claude 給的「絕對核心 5 鏡」這欄。一人接案的現場時間永遠不夠，**先把核心 5 鏡拍完整再拍 nice-to-have**。

骨架還是要 12 鏡，但執行優先序要清楚。剪輯時如果某幾鏡素材不夠好，回頭看「絕對核心」清單，那些不能省的補拍，其他的可以用既有 b-roll 替代。
