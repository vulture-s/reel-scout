# Hook 反推結構 Prompt（拆爆款 reel 用）

> **用途**：拿到一支爆款 reel／TikTok／IG short，要拆出「為什麼會火 + 可遷移到自己題材的結構骨架」。不是抄 hook 金句，是抄 hook 邏輯。
>
> **比直覺看片的價值**：直覺看片大腦會被內容吸走，記得「好看」忘記「為什麼好看」。這個 prompt 強迫拆成可 transfer 的結構單元。
>
> **真實骨架**：Liquid Death（飲料品牌，GMV >10 億美金）廣告 reverse-engineered 的 4-beat 通用骨架 — **Hook（pattern-interrupt）→ 反差介紹 → 清晰 CTA → 結尾餘韻**。不是飲料專屬，可以套幼兒園、咖啡店、線材廣告、課程招生。

---

## Prompt（複製貼上用）

```
我要拆一支爆款短影音的結構，目的是把骨架抽出來套到自己要拍的 [題材] 上。

這是這支 reel 的資訊：
- 平台 / 連結：[IG Reel / TikTok / YouTube Short URL 或描述]
- 內容描述：[2-3 句白話描述影片發生什麼事]
- 互動數據：[讚 / 留言 / 分享，如果有的話]
- 我看完的直覺反應：[一句話，為什麼會停下來看完]

請按「Liquid Death 4-beat 通用骨架」拆解：

1. **Hook（pattern-interrupt）** — 開頭幾秒做了什麼事讓大腦判斷「這不對勁，得繼續看」？
   - 是反預期？是視覺奇觀？是語氣反差？是資訊缺口？
   - 強度給 1-5 分（1 = 普通開場，5 = 完全停下手）

2. **反差介紹** — 進入主題的方式跟一般同類影片差在哪？
   - 一般做法：[列出該題材通常怎麼介紹]
   - 這支做法：[列出這支怎麼介紹]
   - 反差製造的記憶錨點是什麼？

3. **CTA（行動呼籲）** — 影片要觀眾做的下一步是什麼？
   - 明確程度給 1-5 分
   - CTA 跟主題的對齊度（CTA 前後鋪陳是否服務這個動作）

4. **結尾餘韻** — 最後 2-3 秒留下什麼？
   - 笑點？反轉？品牌印象？情感記憶？
   - 跟主題的綁定關係

最後給我「可遷移到 [我的題材] 的骨架版本」：
- 我的 Hook 該長什麼樣（不是抄這支，是用同樣邏輯）
- 我的反差介紹該怎麼設計
- 我的 CTA 要明確到什麼程度
- 我的結尾餘韻該留什麼

審計重點：
- 不要直接抄原片金句或視覺，要抄結構邏輯
- 「反差」要對應到我的題材的真實對立面，不是強行套
- 如果這支 reel 的 Hook 在我的題材不適用，明確告訴我並提供替代結構
```

## Prompt B：已上傳螢幕錄影版（Layer B 用）

> **用途**：沒裝 Reel Scout 的人，直接螢幕錄影一段 reel 上傳給 AI（Gemini 原生看片／Claude 抽幀算特徵），再下這個 prompt。比 Prompt A 多問「節奏」「音訊」兩塊，並加「防腦補＋舉證」護欄。
>
> **為什麼要這版**：2026-06-22 拿世界炸雞「60份雞排」reel 實測——open-ended 問，Gemini/Claude 都漏旁白、把節奏講成「均速」、把加速看成慢動作。改問＋加護欄後，節奏／旁白／加速／觀察-推論分離全部修對。殘留「卡點」「招牌看錯字」是感知型錯誤，靠舉證護欄逼它收斂、最後仍要人回去核。

```
我上傳了一支爆款短影音的螢幕錄影。請你直接看畫面＋聽聲音，把它「為什麼會火 ＋ 可遷移的結構骨架」拆出來，我要套到自己要拍的 [填你的題材] 上。

先說規矩（很重要）：
- 只講你「真的在這支影片看到／聽到」的。看不到、聽不清就明說「不確定」，不要用「短影音通常會…」來補。
- 把「觀察」（畫面/聲音裡的事實）和「推論」（你的解讀）分開標清楚。
- 講「卡點」要舉證：指出哪個動作／剪接對齊哪一拍，並判斷是「刻意設計」還是「剛好時間重合」；分不出來就說「只是時間重合」，不要直接叫卡點。
- 畫面裡的店名／招牌／品牌字，逐字讀出你「實際看到的字」，不要替換成你知道的相似品牌名。

請拆這幾塊：

A. 敘事結構（4-beat）
1. Hook（pattern-interrupt）— 開頭幾秒做了什麼讓人停下來？強度 1-5
2. 反差介紹 — 進主題的方式跟一般同類差在哪？記憶錨點是什麼？
3. CTA — 要觀眾做的下一步？明確度 1-5
4. 結尾餘韻 — 最後 2-3 秒留下什麼？

B. 節奏（逐個講，不要用「平均幾秒一刀」帶過）
- 列出你看到的每個剪接點時間，以及整體是「前快後慢／均速／越來越快」哪一種
- 字幕／字卡換字的頻率，跟剪接點比是快還慢
- 有沒有慢動作／加速／定格，在哪一段

C. 音訊（這支聲音裡到底有什麼）
- 有沒有旁白／口白？有的話把逐字打出來
- 背景音樂是什麼類型？
- 有沒有「卡點」（畫面或動作跟音樂節拍對齊）？還是節奏其實是字幕／剪接帶的？（依上面的舉證規矩）
- 有沒有環境音／音效？
- 哪些是你「真的聽到」、哪些只是「從特徵推測」，分開講

最後：給我可遷移到 [我的題材] 的骨架版本——Hook／反差／CTA／餘韻各該怎麼設計，用邏輯不抄金句。
```

**用法提醒（給人，不在 prompt 裡）**：AI 答完，**節奏和音訊那兩塊回去對著時間軸／打開喇叭核一次**——實測證明它在這兩塊最會自信講錯（卡點、招牌字）。

## 使用範例（音響線材廣告套 Liquid Death 骨架）

### Input

```
我要拆一支爆款短影音的結構，目的是把骨架抽出來套到自己要拍的 [音響線材廣告] 上。

這是這支 reel 的資訊：
- 平台 / 連結：Liquid Death 山泉水「活死人變美國隊長」廣告
- 內容描述：早上 6 點，兩個活死人狀態的人在沙發上奄奄一息；其中一人喝了 Liquid Death，瞬間變身美國隊長，背景換成重金屬音樂。
- 互動數據：YouTube 千萬觀看
- 我看完的直覺反應：本來以為是恐怖片，結果是賣水的，反差感讓我笑出來。
```

### Expected Claude Output（理想版）

```
4-beat 拆解：

1. Hook（pattern-interrupt）— 強度 5/5
   - 開頭 0-3 秒：兩個活死人 + 死寂環境音 + 灰冷色調，觀眾大腦第一反應「這是恐怖片預告？」
   - 為什麼有效：飲料廣告開頭通常是「陽光 + 清新」，這支用恐怖片視覺語言開場 = 強 pattern-interrupt。
   - 資訊缺口：到底要賣什麼？必須看下去才知道。

2. 反差介紹
   - 一般飲料廣告做法：陽光下美女喝水 + 清涼特寫 + 產品 logo 居中。
   - 這支做法：喝產品 = 從活死人「上線」變美國隊長，整段 transformation 用色溫從冷藍灰切到暖飽和 + BGM 從死寂切到重金屬。
   - 記憶錨點：「Liquid Death = 上線狀態切換」這個關聯被植入，比「Liquid Death = 健康水」更具體可記。

3. CTA — 明確程度 3/5
   - 結尾沒有「立刻購買」的硬 CTA，但有產品罐子大特寫 + slogan「Murder Your Thirst」。
   - 對齊度高：整段「上線」敘事在最後 logo 出現時收斂 — 你想上線你就喝這個。

4. 結尾餘韻
   - 美國隊長狀態的角色看鏡頭點頭，定格 1 秒。
   - 留下「我也想試試」的情感記憶 + 品牌個性（不正經、有膽量）的強印象。

可遷移到音響線材廣告的骨架：

Hook 設計：開頭 3 秒拍一個音響玩家「無感聽音樂」狀態 — 戴耳機坐沙發、表情木然、背景靜音、低飽和度。觀眾以為是憂鬱片廣告。
（不是抄活死人，是抄「視覺/音訊基線拉到極低」這個邏輯）

反差介紹：把線材插上系統的瞬間 — 色溫從冷拉到暖、音樂瞬間有「空間感」、玩家眼神「打開」。一般 hi-fi 廣告會放規格表，這支放「狀態切換」。

CTA：不需要「立刻購買」，但要有產品特寫 + 一句 hook line（例如「對抗無感的武器」）。明確程度 3/5 跟原片一致 — hi-fi 觀眾對硬推銷會反感，需要保留品味感。

結尾餘韻：玩家輕輕點頭、閉眼、嘴角微揚的特寫。讓觀眾意識到「這不是規格，是體驗的差距」。

可遷移邏輯總結：「狀態切換」這個敘事單元（從 A 狀態 → B 狀態，產品是 catalyst）比「規格對比」「使用情境」更有記憶點。任何「產品改變使用者狀態」的題材都可以套這個骨架。
```

## 何時用這個 Prompt

**最有 ROI 的時機**：
1. **接案前研究參考片** — 業主給你 reference reel，你要快速抽出可交付方案
2. **拓展新題材** — 自己沒拍過的領域，先拆 3-5 支爆款摸結構規律
3. **自己過去作品複盤** — 用這個骨架反向審視自己拍過的東西為什麼沒火

**不要在這個階段用**：
- 還沒看過影片就請 Claude 拆 — Prompt 需要你先看一次給直覺反應
- 拆同一支拆超過 3 次 — 第三次以後是過度詮釋，挑下一支

## 進階變體

### Variant 1：多支同題材爆款的共同骨架

```
我要找出 [題材] 領域爆款的「共同結構」，不是單一一支。

這是 3-5 支同題材爆款的 4-beat 拆解結果：
[貼上每支的 4-beat 分析]

請告訴我：
1. 哪些 beat 是「全部都有」的（共同骨架）
2. 哪些 beat 是「只有某些做」的（差異化空間）
3. 這個題材的「進入門檻 hook」是什麼樣的（新人拍要過這關才有機會）
4. 這個題材的「天花板招」是什麼樣的（已有人用但執行門檻高）
```

### Variant 2：反向骨架審計（自己企劃 vs 爆款骨架）

```
我自己企劃中的 reel 4-beat 草稿：
- Hook：[描述]
- 反差介紹：[描述]
- CTA：[描述]
- 餘韻：[描述]

這個題材的爆款共同骨架（從上面 Variant 1 整理出來的）：
[貼上共同骨架]

請告訴我：
1. 我的草稿哪些 beat 符合爆款骨架 ✅
2. 哪些 beat 沒做到爆款的水準（具體缺什麼）
3. 哪些 beat 我做了原創嘗試（評估會 work 還是會死）
```

## 跟其他 Prompt 的關係

| Prompt | 階段 | 跟 Hook 反推結構的關係 |
|--------|------|--------------------|
| **Hook 反推結構**（本檔） | 研究 | 從別人的爆款抽結構 |
| **劇本拆解** | 企劃 | 把抽出來的結構翻成自己題材的鏡頭數 / 節奏 / 視角 |
| **焦點審計** | 企劃 | 確認自己的企劃對齊主推焦點 |
| **分鏡視覺化** | 視覺化 | 把確認過的企劃變成可生成的分鏡 prompt |

順序：**Hook 反推（研究）→ 劇本拆解（企劃）→ 焦點審計（守門）→ 分鏡視覺化（執行）**

## 真實案例提示

一支音響訪談 reel 的 3 支參考片（小建炸雞 hook 邏輯 + Eric Luis 拍攝技巧 opening + edocolala 慢節奏訪談結構），如果當時用這個 prompt 拆，會直接抽出「收藏品 → 系統 → 主推產品」這個 progressive 結構，不會等到後期才意識到最後一題失焦。

**4 層信號可靠度提醒**：拆 reel 之前先用 Reel Scout 跑（或沒 Reel Scout 用 3 層 reference 輸入梯度），先拿到 VLM 視覺真相再來 4-beat。**只看 caption 拆 hook 會被誤導**（Eric Luis caption 完全沒提教學內容，但 VLM 看出是 coffee tutorial）。
